## English

### Overview
CyberBlast is an interactive hand gesture-controlled web game featuring cyberpunk aesthetics. Use your webcam and hand movements to play three exciting game modes with stunning particle effects and immersive audio.

### Tech Stack
- **Three.js**: 3D rendering and particle systems
- **MediaPipe Hands**: Real-time hand tracking and gesture recognition
- **TypeScript**: Type-safe development
- **Vite**: Fast development and build tooling
- **Web Audio API**: Immersive sound effects

### Game Modes

#### ğŸ”« Gun Mode
**How to Play:**
1. Point your index finger like a gun
2. Aim at the targets in front of you
3. Make a quick forward motion to shoot
4. Watch the explosive particle effects when you hit targets!

**Gesture:** Index finger extended, other fingers closed

#### ğŸ€ Basketball Mode
**How to Play:**
1. Make a throwing gesture with your hand
2. The power is based on your hand movement speed
3. Faster movements = stronger throws
4. Score by getting the ball through the hoop
5. Enjoy the rainbow explosion when you score!

**Gesture:** Open palm throwing motion

#### ğŸ­ Whip Mode
**How to Play:**
1. Make a fist with your hand
2. Move your fist horizontally back and forth quickly
3. A soft, flowing whip trail will follow your hand
4. Hit the targets with your whip to trigger electric effects!

**Gesture:** Closed fist with fast horizontal movement

### Controls
- **Number Keys (1-3)**: Switch between game modes
  - `1` - Gun Mode
  - `2` - Basketball Mode
  - `3` - Whip Mode
- **M Key**: Toggle music on/off
- **Camera**: Required for hand tracking

### Installation

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

### Features
- Real-time hand gesture recognition
- Cyberpunk particle effects with rainbow colors
- Dynamic audio system with spatial sound
- Smooth camera shake and visual feedback
- Multiple game modes with unique mechanics
